# nobel-prize-data-analysis
An exploratory data analysis of Nobel Prize winners using Python. This project uncovers trends, patterns, and insights across countries, categories, and time using data visualization and statistical techniques.

# 🏆 Nobel Prize Data Analysis

This project explores the history of Nobel Prize winners through data analysis and visualization. The goal is to uncover trends, patterns, and insights across different categories, countries, and time periods.

---

## 📊 Project Overview

The Nobel Prize is one of the most prestigious global awards. In this project, we analyze historical Nobel Prize data to answer key questions such as:

- Which countries have won the most Nobel Prizes?
- How has the number of awards changed over time?
- Which categories dominate in different periods?
- What trends exist in gender representation?
